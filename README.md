## 🚀 Getting Started

### Play Online
Visit **[https://bodhiprotocol.github.io/snake-game/](https://bodhiprotocol.github.io/snake-game/)**

### Play Offline
```bash
# Clone the repo
git clone https://github.com/BodhiProtocol/snake-game.git

# Open in browser
# Double-click index.html  OR  right-click → Open with Browser
```

### How to Play
1. Press any **arrow key** (or tap an on-screen button) to start
2. Guide the snake to eat the red food
3. Each food eaten grows the snake and adds 1 point
4. Avoid hitting the **walls** or your own **tail**
5. Beat your best score!
