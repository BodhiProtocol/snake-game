## [1.0.0] - 2026-06-21

### Added
- Classic Snake game playable in any browser — no installation required
- Arrow key controls + on-screen directional buttons for mobile/touch
- Score tracking with session best score
- Game-over on wall collision and self-collision
- Kid-friendly speed (200ms per tick)
- Dark mode support via `prefers-color-scheme`
- Restart button after game over
- Zero external dependencies — single HTML file
- Works fully offline
