# Contributing to Snake Game

Thank you for your interest in contributing! Here's how you can help.

## 🐛 Reporting Bugs

Found a bug? [Open an issue](https://github.com/BodhiProtocol/snake-game/issues) with:

- **Clear title** — What's broken?
- **Description** — How to reproduce it?
- **Environment** — Browser, OS, device
- **Expected vs actual** — What should happen vs what does?

---

## 💡 Suggesting Features

Have an idea? [Open a feature request](https://github.com/BodhiProtocol/snake-game/issues) with:

- **Clear title** — What feature?
- **Description** — Why is this needed?
- **Use case** — Who benefits?
- **Example** — How would it work?

---

## 🔧 Making Changes

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes to `index.html`
4. Test in Chrome, Firefox, Safari, and Edge
5. Test on desktop and mobile
6. Submit a pull request with a clear description

## ✅ Code Standards

- Vanilla JavaScript only — no frameworks or libraries
- Single-file architecture (`index.html`) — keep it self-contained
- Test on both light and dark mode
- Test on mobile (on-screen buttons must work)
- Keep kid-friendly speed (don't make it too fast)

## 🌐 Testing Checklist

- [ ] Game starts on first arrow key / button press
- [ ] Snake grows on eating food
- [ ] Game over on wall collision
- [ ] Game over on self collision
- [ ] Score increments correctly
- [ ] Best score persists within session
- [ ] Restart works after game over
- [ ] On-screen buttons work on mobile
- [ ] Dark mode renders correctly
- [ ] No console errors
